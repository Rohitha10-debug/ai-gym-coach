# AI Gym Coach

An AI-powered fitness coaching application that combines real-time pose estimation, computer vision, and voice guidance to deliver an interactive workout experience.

## Live Demo

* Application: https://ai-gym-coach-4hfaesfhwep93drbgn44g9.streamlit.app/
* Landing Page: https://rohitha-ai-gym-coach.netlify.app/

## Features

* Real-time exercise tracking using computer vision
* Pose detection and repetition counting
* AI-driven voice feedback and workout guidance
* Personalized workout plans
* Workout history and performance analytics
* Interactive Streamlit user interface

## Tech Stack

* Python
* Streamlit
* OpenCV
* MediaPipe
* TensorFlow Lite
* NumPy
* Pandas
